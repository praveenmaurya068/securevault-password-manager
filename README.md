# 🔐 SecureVault – Password Manager

A Google Apps Script-based password manager with a beautiful web UI.

## Features
- Save, view, update & delete passwords
- Master key encryption (SHA-256)
- Password generator
- Category tagging & search
- CSV export
- Auto session lock

## Setup
1. Open Google Sheets
2. Go to Extensions → Apps Script
3. Paste `Code.gs` into the script editor
4. Create a new HTML file named `index2` and paste the HTML
5. Deploy as a Web App
